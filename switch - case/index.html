Switch - case

A declaração switch é usada para realizar diferentes ações baseadas em condições diferentes.

Exercícios da aula.

Usamos a condição switch para selecionar uma linha de código de um bloco de códigos para ser executado, vejamos abaixo:

swtich(expressao) {
case x:
// bloco de código
break;
case y:
// bloco de código
break.
padrão/default:
// code block
}

É assim que funciona:

A expressão do switch é avaliada uma vez.
O valor da expressão é comparado com os valores de cada caso.
Se algum dos resultados corresponder, o bloco associado irá ser executado.
Se nenhum dos resultados corresponderem, o bloco padrão será executado

Exemplos:

O metodo getDay() nos retorna o dia da semana como um número de 0 até 6.

Domingo = 0, segunda = 1, terça = 2...

Esse exemplo usa o número do dia da semana para calcular o nome do dia da semana.

switch (new Date().getDay()) {
case 0:
day = "Domingo";
break;
case 1:
day = "Segunda-feira";
break;
case 2:
day = "Terça-feira";
break;
case 3:
day = "Quarta-feira";
break;
case 4:
day = "Quinta-feira";
break;
case 5:
day = "Sexta-feira";
break;
case 6:
day = "Sábado";
}

A palavra-chave padrão/default

A palavra-chave padrão/default especifica um código para ser executado caso nenhum resultado corresponda à expressão.

switch (new Date().getDay()) {
case 6:
text = "Hoje é sábado";
break;
case 0:
text = "Hoje é domingo";
break;
default:
text = "Estou ansioso pelo próximo fim de semana";

Blocos de código em comum

Em alguns momentos você vai querer diferentes switch/case para executar o mesmo código.

Nesse exemplo, o case 4 e 5 compartilham o mesmo bloco de código, enquanto o 0 e 6 compartilham outro bloco de código.

switch (new Date().getDay()) {
case 4:
case 5:
text = "Logo será fim de semana";
break;
case 0:
case 6:
text = "É fim de semana";
break;
default:
text = "Estou ansioso pelo fim de semana";
}

Detalhes importantes:

Se vários casos corresponderem ao valor do case, o primeiro case será selecionado.

Se nenhum dos casos corresponderem, o programa continuará e executará o padrão/default.

Em caso de não encontrar nenhum padrão/default, o programa continuará para a declaração após o switch.
