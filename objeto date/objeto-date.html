Objeto Date()

Objeto Date() nos permite trabalhar com datas, sejam elas Ano, Mês, Dia, Horas, Minutos e Segundos.

Sintaxe: const d = new Date();

Por padrão, o JavaScript usará o fuso horário do navegador e vai exibir uma data como um texto em string, exemplo:

const d = new Date();

Objetos Date() são estáticos, o horário muda mas o objeto não.

new Date(year, month,...)

o new Date(year, month,...) cria um novo objeto Date() com uma data e hora específicos. 7 números separados por virgula, por exemplo: Ano, Mês, Dia, Hora, Minuto, Segundo, Milissegundo (nessa ordem).

const d = new Date(2018, 11, 24, 10, 33, 30, 0);

Nota: O JavaScript conta os meses de 0 até 11: Janeiro = 0 e Dezembro = 11.

Utilizando 6, 4, 3 ou 2 números:

6 números especificam ano, mês, dia, hora, minuto e segundo

const d = new Date(2018, 11, 24, 10, 33, 30);

5 números especificam ano, mês, dia, hora e minuto:

const d = new Date(2018, 11, 24, 10, 33);

4 números especificam ano, mês, dia e hora:

const d = new Date(2018, 11, 24, 10);

3 números especificam ano, mês e dia:

const d = new Date(2018, 11, 24);

2 números especificam ano e mês:

const d = new Date(2018, 11);

Você não consegue omitir o mês, se adicionar somente um número, o JavaScript vai ler como milissegundo.

Século anterior:

Um ou dois dígitos serão interpretados como 19xx:

const d = new Date(99, 11, 24);

const d = new Date(9, 11, 24);

new Date(dateString)

new Date(dateString) cria um novo objeto data de uma data em forma de string:

const d = new Date("October 13, 2014 11:13:00");

JavaScript armazena as datas como milissegundo

JavaScript armazena as datas como números de milissegundo desde 01 de Janeiro de 1970 às 00:00:00

new Date(milliseconds)

new Date(milliseconds) cria um objeto date como 0 MAIS os milissegundos dados:

01 de Janeiro de 1970 MAIS 100 000 000 000 milissegundos é aproximadamente 03 de Março de 1973:

const d = new Date(100000000000);

01 de Janeiro de 1970 MENOS 100 000 000 000 milissegundos é aproximadamente 31 de Outubro de 1966:

const d = new Date(-100000000000);

Um dia tem aproximadamente 86 400 000 milissegundos:

const d12 = new Date(86400000);
