Utilizando if, else e else if

Usamos o comando if, else e else if para executar diferentes ações com base em diferentes condições. Muitas vezes quando você escreve um código, você quer executar uma ação diferente para uma decisão diferente.

Usamos if para especificar um bloco de código para ser executado, se a condição especificada for verdadeira.

Usamos else para especificar um bloco de código para ser executado, se a mesma condição for falsa.

Usamos else if para espeficar uma nova condição a ser testada, se a primeira condição for falsa.

A declaração if

Sintaxe:
if (condição) {
// bloco do código para ser executado caso a condição for TRUE
}

Exemplo:
if (hora < 12) {
console.log('Bom dia')
}

A declaração else

Sintaxe:
if (condição) {
// bloco do código para ser executado caso a condição for TRUE
} else {
// bloco do código para ser executado caso a condição for FALSE
}

Exemplo:
if (hora < 12) {
console.log('Bom dia')
} else {
console.log('Boa tarde ou noite')
}

A declaração else if

Sintaxe:
if (condição) {
// bloco do código para ser executado caso a condição for TRUE
} else if (condição2) {
// bloco do código para ser executado caso a condição1 for FALSE e a condição2 for TRUE
} else {
// bloco do código par ser executado se a condição1 for FALSE e a condição2 for FALSE }

Exemplo:
if (hora < 12) {
console.log('Bom dia')
} else if (hora < 18) {
console.log('Boa tarde')
} else {
console.log('Boa noite')
}
