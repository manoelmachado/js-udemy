Valores primitivos e valores por referência

Primitivos (imutáveis): - string, number, boolean, undefined, null (bigint, symbol) - Valores copiados
O que são valores imutáveis? São dados que não conseguimos mudar o tipo de valor.

Exemplo:
let nome = 'Luiz';
nome = 'Otávio';
console.log(nome);

Nesse caso, não estamos mudando o tipo do valor primitivo, estamos trocando o valor da variável.

Podemos fazer uma cópia do dado primitivo usando a seguinte sintaxe:
let a = 'A';
let b = a;
console.log(a, b);
nesse caso, a variável a não depende de b, assim como a variável b não depende de a (a não ser para pegar o seu valor), então a variável b vai manter o seu valor independente se o valor da variável a mudar.

Por exemplo:
a = 'Outra coisa';
console.log(a, b);
Alterando o valor de a, não alteramos o valor de b pois foi feito uma cópia do valor de a quando o valor é primitivo.

Referência (mutáveis) - array, object, function - valores passados por referência

Exemplo:
let a = [1, 2, 3];
let b = a;
console.log(a, b);

Como esses dados são passados por referência, eles vão apontar para o mesmo valor na memória então se alterarmos qualquer um dos valores, os dois mudarão.
Exemplo:

let a = [1, 2, 3];
let b = a;
console.log(a, b);
a.push(4);
console.log(a, b);
b.pop();
console.log(a, b);

Verificamos então que os valores de a e b foram afetados com o comando a.push(4), mesmo falando especificamente para alterar somente a variável a e b em nosso comando respectivamente. Nesse caso, não estamos copiando o valor como anteriormente e sim apontando o valor para o mesmo lugar na memória.

Se por acaso quiséssemos COPIAR o valor de a, podemos usar um operador bem simples, veja:

let a = [1, 2, 3];
let b = [...a];

Isso também serve para objetos, vejamos:

const a = {
nome: 'Luiz',
sobrenome: 'Otávio'
};
const b = a
a.nome = 'João'; console.log(b);

O valor de b vai apotar para o mesmo lugar que a está apontando na memória, fazendo que qualquer alteração na const a, seja aplicada na const b.

Caso queiramos que o valor seja copiado, devemos fazer da seguinte maneira:

const a = {
nome: 'Luiz',
sobrenome: 'otávio'
};
const b = {...a}
a.nome = 'João'; console.log(b);

Então, nesse caso, independente da alteração feita na const a, a const b vai ter o mesmo valor que a const a no início.
