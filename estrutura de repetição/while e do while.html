O loop While

O loop while faz uma rotação pela linha de código desde que uma condição específica seja TRUE

Sintaxe
while (condição) {
// linha de código
}

Exemplo:

while (i < 10) {
texto += "O número é" + i;
i++;
}

Nota: não se esqueça de acrescentar a variável utilizada na condição, caso contrário o loop não irá acabar!
