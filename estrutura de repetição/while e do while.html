O loop While

O loop while faz uma rotação pela linha de código desde que uma condição específica seja TRUE

Sintaxe
while (condição) {
// linha de código
}

Exemplo:

while (i < 10) {
texto += "O número é" + i;
i++;
}

Nota: não se esqueça de acrescentar a variável utilizada na condição, caso contrário o loop não irá acabar!

O loop do... while

Definição e modo de uso:

A declaração do... while define uma linha de código para ser executada uma vez, e após isso se repetirá enquanto a condição for true.

A declaração do... while é comumente usada quando você pretende rodar essa linha de código ao menos uma vez.

Nota: Se você utilizar uma variável na conduição, você deve inicialziar a variável antes do loop e incrementa-la dentro do loop. De outra maneira o loop não vai acabar e irá quebrar seu código/navegador.

Sintaxe:

do {
linha de código para ser executada
}
while (condição);

Exeplo:

let i = 0
while (i < 10) {
console.log(`O número é ` + i)
i++;
