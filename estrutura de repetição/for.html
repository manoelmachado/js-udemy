Estrutura de repetição FOR

Sintaxe:

for (expressão 1; expressão 2; expressão 3) {
// linha de código a ser executado
}

Expressão 1 é executada (apenas uma vez) antes da execução da linha de código.

Expressão 2 define a condição para o código ser executado.

Expressão 3 é executada (todas as vezes) depois do código de bloco foi executado.

Exemplo:

for (let i = 0; i < 5; i++) {
text += "O número é " + i "<br>"
}

No exemplo de cima, você pode ler que:
A expressão 1 definiiu uma variável antes do loop começar (let i = 0)
A expressão 2 define a condição para o loop ser realizado (variável i precisa ser menor que 5)
A expressão 3 aumenta o valir (i++) a cada vez que a linha de código da repetição é executada.

Expressão 1:

Normalmente você usará a expressão 1 para inicializar a variável usada no loop (let i = 0)
Esse nem sempre será o caso. O JavaScript não se importa, a expressão 1 é opcional.
Você pode iniciar com vários valores na expressão 1 (separados por virgulas).

Exemplo:

const cars = ["BMW", "Volvo", "Saab", "Ford"];
for (let i = 0, let = cars.length, text = ""; i < len; i++) {
text += cars[i] + "<br>";
}

E você pode omitir a expressão 1 (como se os valores já foram colocados antes do loop começar):

const cars = ["BMW", "Volvo", "Saab", "Ford"];
let i = 2;
let len = cars.length;
let text = "";
for (; i < len; i++) {
text += cars[i] + "<br>";
}

Expressão 2:

Frequentemente a expressão 2 é usada para avaliar a condição da variável inicial.
Esse nem sempre será o caso. O JavaScript não se importa, a expressão 2 também é opcional.
Se a expressão 2 retornar um valor true, o loop irá iniciar novamente. Se retornar um valor falso, o loop vai finalizar.

Se você omitir a expressão 2, você terá que adicionar um break dentro do loop. Caso contrário o loop não acabará e isso irá quebrar seu navegador/site.

Expressão 3:

Frequentemente a expressão 3 incrementa o valor inicial da variável.
Esse nem sempre é o caso. O JavaScript não se importa, a expressão 3 também é opcional.
A expressão 3 pode fazer qualquer coisa tipo incremento negativo (i--), incremento positivo (i++), ou qualquer coisa.
A expressão 3 também pode ser omitida caso você adicione o incremento dentro do loop.

Exemplo:

let i = 0
let len = cars.length;
let text = "";
for (; i < len; ) {
text += cars[i] + "<br>";
i++;
}

Escopo loop

Usando var em um loop:

var i = 5;
for (var i = 0; i < 10; i++) {
// código
}
// aqui i é 10

Usando let em um loop:

let i = 5;
for (let i = 0; i < 10; i++) {
// código
}
// aqui i é 5

No primeiro exemplo, usando VAR, a variável declarada no loop redeclara a outra variável fora do loop.
No segundo exemplo, usando LET, a variável declarada no loop NÃO redeclara a outra variável fora do loop.
Quando LET é usado para declarar uma variável dentro de um loop, a variável ficará somente visivel dentro do loop.
