For in

A declaração for in no JavaScript faz um loop através das propriedades de um objeto:
Sintaxe
for (key in object) {
// linha de código para ser executado
}

Exemplo:

const pessoa = {fname:"Manoel", lname:"Machado", age:30};
let text = "";
for (let x in pessoa) {
text += pessoa[x];
}

Explicando o exemplo:
O loop for in itera(repete) sobre o objeto pessoa
Cada iteração retorna a chave (x)
A chave é utilizada para acessar o valor da chave
O valor da chave é pessoa[x]

For in em Arrays

A declaração for in também faz loop sobre as propriedades de um Array:

Sintaxe
for (variavel in array) {
// código a ser executado
}

Exemplo:
const numeros = [45, 4, 9 16, 25];
let texto = "";
for (let x in numeros) {
texto += numeros[x];
}

Nota: Não use o FOR IN em um Array se a ordem do indice for importante.
A ordem do índice é dependente da implementação, e os valores do array podem não ser acessados na ordem que você esperava.
É melhor usar um loop for, um loop for of, ou Array.forEach() quando a ordem for importante.

Array.forEach()

O método forEach() chama uma função (uma função callback) uma vez para cada elemento do array.

Exemplo
const numeros = [45, 4, 9 16, 25];
let texto = "";
numeros.forEach(myFunction);

function myFunction(valor, index, array) {
texti += valor;
}

Note que a função tem 3 argumentos:
O valor do item
O index do item
E o array em si

A função acima usa somente o parametro valor. Ela pode ser reescrita assim:

const numeros = [45, 4 9, 16, 25];
let texto = "";
numeros.forEach(myFunction);

function myFunction(value) {
texto += valor;
}
