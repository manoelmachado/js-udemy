For Of

A declaração For Of no JavaScript faz um loop através dos valores de um objeto iterável.

Ele te permite realizar loops através de dados de estrutura iterávels como Arrays, Strings, Maps, NodeLists e mais:

Sintaxe:
for (variavel of iterável) {
// código para ser executado
}

variável - Para cada iteração no valor da próxima propriedade é atribuído para a variável. A variável pode ser declarada com const, let ou var.

iterável - Um objeto que tem propriedade iterável.

Looping em um Array

const carros = ["BMW", "Volvo", "Mini"];

let texto = "";
for (let x of carros) {
texto += x;
}

Looping em uma String

let linguagem = "JavaScript";

let texto = "";
for (let x of linguagem) {
texto += x;
}
