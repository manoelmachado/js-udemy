Operação ternária

Operação ternário é o único operador JavaScript que possui três operandos. Esse operador é frequentemente usado como um atalho para a instrução if.

Sintaxe:

condição ? expr1 : expr2

Parâmetros

condition: Uma expressão que é avaliada como true ou false.

expr1, expr2: expressões com valores de qualquer tipo.

Descrição

Se condition é true, o operador retornará o valor de expr1; se não, ele retornara o valor de expr2. Por exemplo, para exibir uma mensagem diferente baseada no valor da variável isMember, você pode utilizar o seguinte código:

"The fee is " + (isMember ? "R$2,00" : "R$10,00")

Conforme o resultado da operação, você também poerá atribuir a variáveis:

var elvisLives = Math.PI > 4 ? "Yep" : "Nope";

Também são possíveis múltiplas avaliações ternárias (nota: o operador condicional é associativo a direita):

var firstCheck = false,
secondCheck = false,
access = firstCheck ? "Access denied" : secondCheck ? "Acess denied" : "Access granted";

console.log(access); // logs "Acess granted"

Você também pode usar avaliações ternárias no espaço livre de modo a fazer diferentes operações:

var stop = false, age = 16;

age > ? location.assign("continue.html") : stop = true;

Você também pode fazer mais do que uma única operação em cada caso, serparando-os por virgula:

var stop = false, age = 13;
age > 18 ? (
alert("Ok, you can go"),
location.assign("continue.html")
) : (
stop = true,
alert("Sorry, you are much too young!")
);
